# 🛸 Drone Ascension

Браузерный 3D-прототип воздушного экшена: пилотируй дрон, собирай энергетические сферы магнитным притяжением, качай уровень, свети ярче — становись целью, сражайся с ботами, роняй энергию при гибели и возрождайся. Вдохновлено связкой Agar.io + 3D-шутер.

Вся игра — **один самодостаточный HTML-файл** (`index.html`). Единственная зависимость — Three.js r128, подгружаемый с CDN. Никакой сборки, никаких node-модулей: открыл в браузере — играешь.

> ⚠️ Это ранний прототип. Противники — боты (реального сетевого мультиплеера пока нет). Управление — только ПК (клавиатура + мышь).

---

## 🎮 Управление

| Клавиша | Действие |
|---|---|
| `W A S D` | тяга (вперёд/назад/вбок) |
| `Мышь` | обзор и прицел (pointer-lock) |
| `ЛКМ` | огонь |
| `Space` / `Ctrl` | вверх / вниз |
| `Shift` | форсаж |
| `Q` | рывок (Dash) |
| `1 · 2 · 3` | выбор улучшения на ходу |
| `M` | звук вкл/выкл |
| `Esc` | пауза |

## ✨ Что есть в прототипе

- **Полёт от третьего лица** с креном корпуса, форсажем и рывком.
- **8 типов сфер**, каждая со своим цветом, формой, иконкой и эффектом: энергия (опыт), боевая (буст урона), щит, ремонт, перегрузка, магнит, ускоритель, легендарная (радужная, оповещает всю карту).
- **Система улучшений**: каждые 5 уровней — выбор 1 из 3 (двигатель / оружие / щиты / корпус / энергия / сбор), 18 апгрейдов.
- **Визуальная эволюция дрона**: улучшения меняют модель (сопла, излучатели, энергокольцо, массивность корпуса), сила видна по внешнему виду.
- **Мультяшный cel-shading**: яркое дневное небо, тёплое солнце, toon-шейдинг, контуры «свой/чужой», парящие травяные острова, облака.
- **Синтезированный звук** (Web Audio, без файлов): двигатель, ветер, подбор, удары, лазер, музыка в меню.
- **Боты с ИИ** (поиск / охота / бегство / сбор добычи), таблица лидеров, индикатор угрозы, 15-минутный матч.

## 🚀 Как запустить локально

Файл статический, поэтому достаточно любого локального веб-сервера (открытие через `file://` тоже обычно работает, но сервер надёжнее для pointer-lock и звука).

```bash
# Вариант 1 — Python (ничего ставить не нужно)
python3 -m http.server 8080
# открой http://localhost:8080

# Вариант 2 — Node
npx serve .
```

## 🌐 Деплой на GitHub Pages

Игра лежит в корне как `index.html`, поэтому Pages раздаёт её напрямую.

**Быстрый способ:** Settings → Pages → Source: *Deploy from a branch* → ветка `main`, папка `/ (root)`. Через минуту сайт будет доступен по адресу `https://<логин>.github.io/<репозиторий>/`.

**Через Actions:** в репозитории уже лежит `.github/workflows/deploy.yml` — он публикует сайт автоматически при каждом пуше в `main`. Нужно только один раз выбрать Settings → Pages → Source: *GitHub Actions*.

Файл `.nojekyll` отключает обработку Jekyll, чтобы всё раздавалось как есть.
